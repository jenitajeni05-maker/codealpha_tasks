Food Delivery App UX Project
Wireframe | UI Design | Zomato UX Case Study

Project Overview
This repository contains three UX/UI tasks completed as part of a design project.
The work progresses from wireframing → UI design → UX case study analysis, following a proper UX design workflow

Project Structure (Reordered)

Type: Low-fidelity wireframe
Purpose:
Define screen layout and structure
Plan user flow and navigation
Reduce clutter and improve usability
Key Screens:

Cart
tracking
Tool Used: Figma

2️⃣ UI Design (Task 2)
Type: High-fidelity UI design
Purpose:
Convert wireframes into visually appealing screens

Improve readability and accessibility
Highlights:
Clean and minimal interface
Clear price hierarchy

User-friendly layout
 Tool Used: Figma

Conclusion
This project demonstrates a complete UX/UI process starting from wireframe planning, moving to UI design, and finally validating decisions through a real-world UX case study (Zomato).
The outcome focuses on improved usability, accessibility, and a smoother user experience.
